# nodes
points in a networking. They include: 
- routers
	- provide connectivity between LANs
	- Example: Cisco ISR
- switches
	- route traffic inside LANs
- physical firewalls
	- can block traffic using configured set of rules
- server
	- host services which are accessed by clients
- end hosts / clients:
	- access the services of a server
	- smartphones
	- laptops
	- PCs
	- etc.
