# Inheritance in OOP
Inheritance is receiving fields and properties from one object. These receiving objects are called child objects.

For example if the class worker has 2 fields , salary and title. You can create a child of that class, we will call it special_worker , that inherits the salary and title field of its parent of you can also create a new field only for the fields inherent to the child class. For our child class, we will give it a field called, tenure. So the special_worker class has 3 fields: title, salary, tenure. While the worker class has 2 : title and salary.

The parent class is called the base class and the child class is called the derived class or extended class.

the child class (aka: subclass, derived class, extended class) can be thought of as a "more specialized" version of the parent class. It is more specific in the actions it does then the parent class.
