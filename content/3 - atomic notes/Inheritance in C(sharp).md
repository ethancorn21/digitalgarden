# Inheritance in C(sharp)
```cs
// child will inherent all of the methods and fields of the parent class
class parent{
	void exampleMethod(){
		console.log("This method does nothing")
	}
}

class child : parent{
	void anotherExample(){
		console.log("This is the second method inside this class")
	}
}
```
