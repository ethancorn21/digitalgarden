initialize git in whatever folder you are working in.
```shell
git init
```

```shell
git config user.name "**insert name**"
git config user.email "**insert email**"
```

```shell
git status          # Check what changed
git add .           # Stage everything
git commit -m ""    # Create commit
git log --oneline   # See your progress
git restore .       # Undo if you mess up
```
