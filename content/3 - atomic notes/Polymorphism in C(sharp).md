# Polymorphism in C(sharp)

polymorphism is a way to write code so that you can write a standardized action, but that action changes based on the object it is called against. For example if i called a process payment method, the exact means of that process would change based on if it was called against a debit or credit card object. Polymorphism tries to standardize actions to promote reusability, often times it implements interfaces which are like "contracts" that state, if you use this interface you must have this method.
