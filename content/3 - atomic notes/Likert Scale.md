# Likert Scale
A way to measure unquantifiable things (feelings, opinions, etc.) using a numbered scale.

### example
On a scale of 1-5 (1 = very unsatisfactory, 5 = satisfactory), how did you enjoy reading this card about the Likert Scale?
1) very unsatisfactory
2) unsatisfactory
3) neutral
4) satisfactory
5) very satisfactory
