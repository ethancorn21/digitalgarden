Slices are pieces of a list. formatted as:
``list[first index : index to end at]

```python
my_list = [1,2,3,4,5]
print(my_list[0:3]) # prints 1 2 3 but not 4 because it ends @ index 3
print(my_list[:])   # prints all index from beginning to end
print(my_list[:2])  # prints from index 0 and stops @ index 2
```
