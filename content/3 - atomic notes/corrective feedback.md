# corrective feedback
feedback that acts as course correction to let you know if what you are doing is right. For students this looks like grades, which give you feedback if your learning and study methods are working. For workers, this can look like performance evaluations. It can also be if something works the way you want it to, asking yourself if there is a better way, many different ways to solicit feedback.
