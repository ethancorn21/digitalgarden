# Attribution of Consciousness and The False Consensus Effect

do we attribute consciousness to animals and others things because of the false consensus effect? all we have ever known is consciousness and so it is easy for us to attribute that to others because it is hard for us to imagine being in any state other than conscious. In fact, it is probably impossible to imagine any other state. If consciousness is all any of us have ever know, it is the only state of being that we can contribute to other things. Things like animals or artificial intelligence that seems as though they live and make decisions as we do.
