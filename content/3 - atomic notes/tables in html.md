tables consists of rows and columns

```html
<table>
	<tr>
		<!-- <tr> creates a new row -->
	
		<th>header 1</tr>
		<th>header 2</tr>
		<!-- <th> is a table header-->
	</tr>
	
	<tr>
		<td>tabledata1</td>
		<td>tabledata2</td>
		<!-- <td> creates new table data -->
	</tr>
</table>

```
