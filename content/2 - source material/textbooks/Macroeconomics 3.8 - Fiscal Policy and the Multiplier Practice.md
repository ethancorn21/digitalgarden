[[macroeconomics]]


1 / however much you save * 100  = multiplier
then gap / multiplier = needed spending

If government cut taxes, that would then be affected by MPS and MPC as well.
If MPC is .5, and there is a $40 billion deficit, how much would the government need to change taxes by?
multiplier = 1/.5 = 2.
You might think the government should cut taxes by $20 billion but if they did, only $10 billion would be saved because the new money is again subject to MPS and MPC.

If MPC is 0.8, the deficit is $40 billion, how much should the government change taxes by?
multiplier = 1/.2 = 5
$40 billion /5  = 8 billion.
The government needs $8 billion.
if the government changed taxes by $8 billion, only 4/5 of $8 billion would be spent. But, the government needs $8 billion spent, so the answer is to change taxes so that 4/5 of whatever it is = $8 billion. 4/5th = 8/10. So the answer is to change taxes by $10 billion

The easy way to calculate = MPC / MPS = multiplier
.8/.2 = 4. Deficit = $40 billion. 40 / 4 = 10. So $10 billion dollar change needed
