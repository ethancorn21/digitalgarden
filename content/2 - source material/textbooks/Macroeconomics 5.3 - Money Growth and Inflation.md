- [[macroeconomics]]


Money market graph = nominal interest rate - short run decision making
loanable funds graphs = real interest rate - long run decision making

monetary policy = increase money supply increases GDP eventually
quantity theory of money = increase to money supply only increases prices not GDP

though contradictory, in short run both coexist



real rate = nominal - expected rate

Fisher Effect - an increase in expected inflation causes nominal interest rates to increase

Natural Rate of Interest - long run trends created by the loanable funds market. 

in the long run, things return back to natural rate. So if short run interest rate increases, long run does not change.
